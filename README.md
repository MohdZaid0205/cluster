## About


<div align="center" font-size="10px" font-family="Cascadia Mono">
<p align="center">
  <strong>Cluster</strong> is a high-performance, community-driven discussion platform.
</p>
<br/>
Unlike standard social networks, Cluster emphasizes structure and credibility. Features like Windows (cluster-references) and Megaphones (time-sensitive announcements) allow for sophisticated content curation. The platform prioritizes reputation and moderation, utilizing both human oversight and automated Bot Rules to maintain professional standards.
</div>

## Tech Stack

- **Framework**: [FastAPI](https://fastapi.tiangolo.com/) (Asynchronous Python Web Framework)
- **ORM**: [SQLModel](https://sqlmodel.tiangolo.com/) (SQLAlchemy + Pydantic)
- **Database**: Core support for SQLite (development) and MySQL (production).
- **Validation**: [Pydantic v2](https://docs.pydantic.dev/) for data integrity and settings management.
- **Testing**: [Pytest](https://docs.pytest.org/) with full integration for API, service, and database triggers.
